# 🎓 Student Placement Prediction System

## Overview
This project is a Machine Learning web application developed using **Python**, **Scikit-learn**, and **Streamlit** to predict whether a student is likely to be placed based on academic performance, technical skills, and placement-related attributes.

## Features
- Student Information Input
- Academic Performance Analysis
- Skills Assessment
- Placement Prediction
- Probability Score
- Career Improvement Suggestions
- Interactive Streamlit Dashboard

## Machine Learning
- Algorithm: Logistic Regression
- Hyperparameter Tuning: GridSearchCV
- Preprocessing: ColumnTransformer + Pipeline

## Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Joblib
